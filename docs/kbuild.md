# `kbuild` In `ktools-kotlin`

The Kotlin workspace currently uses the shared `kbuild` command model for
workspace operations.

## Current Status

- the checked-out workspace does not currently contain a separate `kbuild/`
  implementation directory
- the documented entrypoint is the shared `kbuild` command on `PATH`
- this page exists so the website can show the Kotlin workspace's current
  build-tool state from canonical markdown
